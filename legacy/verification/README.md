# VIVA Verification Lab

**"The Lab"**

This directory is for scientific experiments, recurrence tests, and hardware validation scripts that are NOT standard unit tests.

## Structure
- **`body/`**: Hardware interaction, NIF benchmarks, physics validation.
- **`memory/`**: Vector DB usage, associative memory recall experiments.
- **`protocol/`**: Communication protocols (Lindblad, etc.) validation.

## Usage
Run these scripts with:
```bash
mix run verification/path/to/script.exs
```
